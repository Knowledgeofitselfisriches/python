## git 简介
	  git的优势：
	  	1. 大部分操作本地完成无需联网
	  	2. 完整性保证 hash 操作
	  	3. 尽可能添加数据而不是添加和修改数据
	  	4. 分支操作流畅快捷
	  	5. 兼容linux命令	
	  工作区 暂存区  本地库
	  代码托管中心：维护远程库

	  团队内操作
		  本地库 push--->   github
		  本地库	<---- pull  github
		  本地库 <--- clone github
	  团队外贡献者
	  	  本地库 push--->   github
		  本地库	<---- pull  github
		  本地库 <--- clone github
